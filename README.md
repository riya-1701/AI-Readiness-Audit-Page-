AI Readiness Audit

A simple web tool that analyzes how well a website is optimized for AI systems (like ChatGPT, Gemini, Claude, Grok, etc.) to understand, extract, and use its content.
This project focuses on practical AI-readiness signals, not just traditional SEO.

This tool simulates how AI systems (like search engines or LLMs) interpret a webpage. Users can enter a website URL and receive:
-A score out of 100
-A list of issues impacting AI readability
- Insights into content structure and quality

What is “AI Readiness”?

AI Readiness refers to how easily AI systems can:
-Understand your content
-Extract meaningful information
-Represent your website in search or responses

This tool evaluates 5 basic checks, such as:
-Meta descriptions
-Heading structure (H1, H2)
-Content length
-Image accessibility (alt text)
-Presence of FAQs

Tech Stack:
Frontend: React.js, JavaScript(ES6+)
Backend: Express.js, REST API
